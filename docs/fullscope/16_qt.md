# Qt — `qt.*`

Zcythe provides Qt5/Qt6 widget support through the `@zcy.qt` package. Import it with an alias:

```
@import(qt = @zcy.qt)
```

The build system automatically detects Qt usage, compiles a thin C++ wrapper, and links it. Qt headers must be installed:

- Fedora/RHEL: `sudo dnf install qt5-qtbase-devel` (or `qt6-qtbase-devel`)
- Debian/Ubuntu: `sudo apt install qtbase5-dev` (or `qt6-base-dev`)
- macOS: `brew install qt`

Qt6 is preferred; Qt5 is used as a fallback.

---

## Application Lifecycle

### `qt.app()` — Create the Application

Must be called before creating any widgets. Returns the application object.

```
app := qt.app()
```

### `app.run()` — Run the Event Loop (blocking)

Enters Qt's event loop and blocks until all windows are closed.

```
app.run()
```

### `app.process_events()` / `app.should_quit()` — Manual Loop

For programs that need to react to widget state each frame:

```
while !app.should_quit() {
    app.process_events()
    if btn.clicked() {
        # handle click
    }
}
```

## Window

### `qt.window(title, width, height)` — Create a Window

```
win := qt.window("My App", 800, 600)
```

### Window Methods

| Method | Description |
|--------|-------------|
| `win.show()` | Make the window visible |
| `win.set_layout(layout)` | Set the root layout |
| `win.set_title(title)` | Change the window title |
| `win.resize(w, h)` | Resize the window |

## Widgets

All widget constructors return objects with a `.widget` field (used internally by layouts).

### `qt.label(text)` — Text Label

```
lbl := qt.label("Hello, World!")
lbl.set_text("Updated text")
t   := lbl.text()   # str
```

### `qt.button(text)` — Push Button

```
btn := qt.button("Click me")
if btn.clicked() {
    # triggered once per click, resets after reading
}
btn.set_text("New label")
```

### `qt.input()` — Single-Line Text Input

```
field := qt.input()
field.set_placeholder("Enter your name...")
field.set_text("default")
name := field.text()   # str
```

### `qt.checkbox(text)` — Checkbox

```
cb := qt.checkbox("Enable feature")
if cb.checked() { ... }
cb.set_checked(true)
if cb.changed() { ... }   # true once per state change
```

### `qt.spinbox(min, max)` — Integer Spin Box

```
spin := qt.spinbox(0, 100)
v    := spin.value()        # i32
spin.set_value(42)
if spin.changed() { ... }   # true once per value change
```

## Layouts

Layouts arrange widgets vertically or horizontally. Pass a layout to `win.set_layout(...)`.

### `qt.vbox()` — Vertical Box Layout

```
layout := qt.vbox()
layout.add(lbl)
layout.add(btn)
layout.set_spacing(10)
layout.set_margin(20)
layout.add_stretch()
```

### `qt.hbox()` — Horizontal Box Layout

```
row := qt.hbox()
row.add(lbl)
row.add(field)
```

### Layout Methods

| Method | Description |
|--------|-------------|
| `layout.add(widget)` | Add a widget (anything with `.widget` field) |
| `layout.add(sublayout)` | Add a nested layout (anything with `.layout` field) |
| `layout.add_stretch()` | Add flexible spacer |
| `layout.set_spacing(n)` | Pixel gap between items |
| `layout.set_margin(n)` | Border margin in pixels |

`add` works for both widgets and nested layouts — it detects the type automatically.

## Complete Example: Counter

```
@import(qt = @zcy.qt)

@main {
    app := qt.app()

    win    := qt.window("Counter", 300, 200)
    lbl    := qt.label("0")
    inc    := qt.button("+1")
    dec    := qt.button("-1")
    reset  := qt.button("Reset")

    row := qt.hbox()
    row.add(dec)
    row.add(inc)

    layout := qt.vbox()
    layout.add(lbl)
    layout.add(row)
    layout.add(reset)
    layout.set_spacing(8)
    layout.set_margin(16)

    win.set_layout(layout)
    win.show()

    count := 0
    while !app.should_quit() {
        app.process_events()
        if inc.clicked()   { count += 1 ; lbl.set_text(@str(count)) }
        if dec.clicked()   { count -= 1 ; lbl.set_text(@str(count)) }
        if reset.clicked() { count = 0  ; lbl.set_text("0") }
    }
}
```

## Complete Example: Form

```
@import(qt = @zcy.qt)

@main {
    app := qt.app()

    win   := qt.window("Form", 360, 220)
    name  := qt.input()
    age   := qt.spinbox(0, 150)
    send  := qt.button("Submit")
    out   := qt.label("")

    name.set_placeholder("Full name")

    form := qt.vbox()
    form.add(name)
    form.add(age)
    form.add(send)
    form.add(out)
    form.set_spacing(6)
    form.set_margin(12)

    win.set_layout(form)
    win.show()

    while !app.should_quit() {
        app.process_events()
        if send.clicked() {
            n := name.text()
            a := age.value()
            out.set_text(n)
            @pf("submitted: name={n}  age={a}\n")
        }
    }
}
```

## Quick Reference

| Call | Description |
|------|-------------|
| `qt.app()` | Create application |
| `app.run()` | Blocking event loop |
| `app.process_events()` | Process pending events |
| `app.should_quit()` | True when all windows closed |
| `qt.window(title, w, h)` | Create main window |
| `win.show()` | Show window |
| `win.set_layout(layout)` | Set root layout |
| `win.set_title(t)` | Change title |
| `win.resize(w, h)` | Resize |
| `qt.label(text)` | Text label |
| `lbl.set_text(t)` | Update label text |
| `lbl.text()` | Get label text |
| `qt.button(text)` | Push button |
| `btn.clicked()` | True once per click |
| `btn.set_text(t)` | Change button label |
| `qt.input()` | Text input field |
| `field.text()` | Current input text |
| `field.set_text(t)` | Set input text |
| `field.set_placeholder(t)` | Placeholder hint |
| `qt.checkbox(text)` | Checkbox |
| `cb.checked()` | Current checked state |
| `cb.set_checked(v)` | Set checked state |
| `cb.changed()` | True once per state change |
| `qt.spinbox(min, max)` | Integer spin box |
| `spin.value()` | Current value |
| `spin.set_value(v)` | Set value |
| `spin.changed()` | True once per value change |
| `qt.vbox()` | Vertical layout |
| `qt.hbox()` | Horizontal layout |
| `layout.add(item)` | Add widget or sublayout |
| `layout.add_stretch()` | Add flexible spacer |
| `layout.set_spacing(n)` | Gap between items |
| `layout.set_margin(n)` | Border margin |
