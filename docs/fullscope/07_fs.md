# I/O & File System — `@fs::`

The `@fs::` namespace provides portable file system operations. No import is needed.

---

## API Overview

Zcythe provides four distinct file I/O APIs:

| API | Open with | Use when |
|-----|-----------|----------|
| `@fs::file_reader` | `try @fs::file_reader::open(path)` | Text files — read all, line-by-line, or byte-by-byte |
| `@fs::file_writer` | `try @fs::file_writer::open(path)` | Text files — write strings or lines |
| `@fs::byte_reader` | `try @fs::byte_reader::open(path)` | Binary files with endian-aware integer reads |
| `@fs::byte_writer` | `try @fs::byte_writer::open(path)` | Binary files with endian-aware integer writes |
| `@fs::reader` | `@fs::reader::init(path)` | Simple handle-based reader; returns null if path invalid |
| `@fs::writer` | `@fs::writer::init(path)` | Simple handle-based writer; returns null if path invalid |

---

## Path Utilities

```
if @fs::isFile("config.txt") { @pl("found config") }
if @fs::isDir("src")         { @pl("src directory exists") }

@fs::mkdir("output/logs")           # create directory (and all parents)
@fs::mkfile("output/result.txt")    # create empty file (truncates if exists)
@fs::del("output/temp")             # delete file or directory
@fs::rename("old.txt", "new.txt")   # rename / move
@fs::mov("src/file.txt", "dst/")    # alias for rename
```

## Text File Reading — `@fs::file_reader`

```
fn read_all(path: str) -> any {
    f := try @fs::file_reader::open(path)
    defer f.cl()

    content := try f.rall()
    @pl(content)
}
```

### Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `f.rall()` | `str!` | Read entire file into a string |
| `f.rln()` | `str!` | Read one line (strips trailing `\n`) |
| `f.rch()` | `char!` | Read a single byte |
| `f.r(n)` | `[]u8!` | Read exactly n bytes |
| `f.eof()` | `bool` | `true` while data remains |
| `f.cl()` / `f.close()` | void | Close the file |

### Read Line by Line

```
fn print_lines(path: str) -> any {
    f := try @fs::file_reader::open(path)
    defer f.cl()

    while f.eof() {
        line := try f.rln()
        @pl(line)
    }
}
```

## Text File Writing — `@fs::file_writer`

```
fn save(path: str, content: str) -> any {
    f := try @fs::file_writer::open(path)
    defer f.cl()
    try f.w(content)
}
```

### Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `f.w(data)` | `void!` | Write string or bytes |
| `f.wln(data)` | `void!` | Write string then newline |
| `f.wch(byte)` | `void!` | Write a single byte |
| `f.fl()` | `void!` | Flush to disk |
| `f.cl()` / `f.close()` | void | Close the file |

## Binary File Reading — `@fs::byte_reader`

```
fn read_binary(path: str) -> any {
    f := try @fs::byte_reader::open(path)
    defer f.cl()
    header := try f.r(4)
    @pf("magic: {header[0]:x} {header[1]:x}\n")
}
```

### Endianness Constants

| Constant | Value |
|----------|-------|
| `@fs::Little` | little-endian |
| `@fs::Big` | big-endian |

## Binary File Writing — `@fs::byte_writer`

```
fn write_binary(path: str) -> any {
    f := try @fs::byte_writer::open(path)
    defer f.cl()
    try f.w({0xDE, 0xAD, 0xBE, 0xEF})
}
```

## Directory Listing — `@fs::ls`

`@fs::ls(path)` lists directory contents. Returns `?[]entry` — check against `undef` before use.

```
files := @fs::ls("/home/user/docs")
if files != undef {
    @pf("found {} items\n", files.len)
    for e => files {
        @pf("  {}\n", e.path())
    }
}
```

### Entry Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `e.path()` | `str` | Absolute path of the entry |
| `e.isFile()` | `bool` | True if entry is a regular file |
| `e.isDir()` | `bool` | True if entry is a directory |

---

## Handle Reader — `@fs::reader`

A two-step handle API: create with `init`, then `open`. Returns `?handle` — check against `undef` before use. Does not use `try`/`catch`.

```
rdr := @fs::reader::init("/etc/hostname")
if rdr != undef {
    rdr.open()
    defer rdr.cl()
    @pl("opened ok")
}
```

### Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `rdr.open()` | void | Open the file for reading |
| `rdr.cl()` | void | Close the file |
| `rdr.readLine(buf)` | `?[]u8` | Read next line into buf; null at EOF |

---

## Handle Writer — `@fs::writer`

```
wtr := @fs::writer::init("/tmp/output.txt")
if wtr != undef {
    wtr.open()
    wtr.write("hello world\n")
    wtr.cl()
}
```

### Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `wtr.open()` | void | Create/truncate the file for writing |
| `wtr.cl()` | void | Close the file |
| `wtr.write(str)` | void | Write a string to the file |

## Complete Example: Word Count

```
fn count_words(path: str) -> i32 {
    f := @fs::file_reader::open(path) catch |_| { _ => { ret 0 } }
    defer f.cl()

    content := f.rall() catch |_| { _ => { ret 0 } }

    count := 0
    in_word := false
    for ch => content {
        if ch == ' ' or ch == '\n' or ch == '\t' {
            if in_word { count += 1; in_word = false }
        } else {
            in_word = true
        }
    }
    if in_word { count += 1 }
    ret count
}

@main {
    n := count_words("README.md")
    @pf("word count: {n}\n")
}
```
