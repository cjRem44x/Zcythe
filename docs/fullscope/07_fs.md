# I/O & File System — `@fs::`

The `@fs::` namespace provides portable file system operations. No import is needed.

---

## Path Utilities

```
# Check if a path exists and is a file
if @fs::isFile("config.txt") {
    @pl("found config")
}

# Check if a path is a directory
if @fs::isDir("src") {
    @pl("src directory exists")
}

# Create a directory (and all parents)
@fs::mkdir("output/logs")

# Create an empty file (truncates if it exists)
@fs::mkfile("output/result.txt")

# Delete a file or directory recursively
@fs::del("output/temp")

# Rename / move a file
@fs::rename("old.txt", "new.txt")
@fs::mov("src/file.txt", "dst/file.txt")    # alias for rename
```

## Text File Reading

Open a file for reading with `@fs::FileReader::open`. The call returns an error union — use `try` or `catch`.

```
fn read_all(path: str) -> any {
    f := try @fs::FileReader::open(path)
    defer f.cl()

    content := try f.rall()
    @pl(content)
}
```

### FileReader Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `f.rall()` | `str!` | Read entire file into a string |
| `f.rln()` | `str!` | Read one line (strips trailing `\n`) |
| `f.rch()` | `char!` | Read a single byte |
| `f.r(n)` | `[]u8!` | Read exactly n bytes |
| `f.eof()` | `bool` | Returns `true` while data remains (inverse: `false` = EOF reached) |
| `f.cl()` / `f.close()` | void | Close the file |

### Read Line by Line

```
fn print_lines(path: str) -> any {
    f := try @fs::FileReader::open(path)
    defer f.cl()

    while f.eof() {
        line := try f.rln()
        @pl(line)
    }
}
```

## Text File Writing

```
fn save(path: str, content: str) -> any {
    f := try @fs::FileWriter::open(path)
    defer f.cl()

    try f.w(content)
}
```

### FileWriter Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `f.w(data)` | `void!` | Write string or bytes |
| `f.wln(data)` | `void!` | Write string then newline |
| `f.wch(byte)` | `void!` | Write a single byte |
| `f.fl()` | `void!` | Flush to disk (sync) |
| `f.cl()` / `f.close()` | void | Close the file |

### Append Lines

```
fn log_line(path: str, msg: str) -> any {
    # createFile truncates; to append you would open differently
    f := try @fs::FileWriter::open(path)
    defer f.cl()
    try f.wln(msg)
}
```

## Binary File I/O

`@fs::ByteReader` and `@fs::ByteWriter` work like their text counterparts but are intended for binary data with explicit endianness control.

### Endianness Constants

| Constant | Value |
|----------|-------|
| `@fs::Little` | little-endian |
| `@fs::Big` | big-endian |

```
fn write_binary(path: str) -> any {
    f := try @fs::ByteWriter::open(path)
    defer f.cl()
    try f.w({0xDE, 0xAD, 0xBE, 0xEF})
}

fn read_binary(path: str) -> any {
    f := try @fs::ByteReader::open(path)
    defer f.cl()
    header := try f.r(4)
    @pf("magic: {header[0]:x} {header[1]:x}\n")
}
```

## Complete Example: Word Count

```
fn count_words(path: str) -> i32 {
    f := @fs::FileReader::open(path) catch |_| { _ => { ret 0 } }
    defer f.cl()

    content := f.rall() catch |_| { _ => { ret 0 } }

    count := 0
    in_word := false
    for ch => content {
        if ch == ' ' or ch == '\n' or ch == '\t' {
            if in_word {
                count += 1
                in_word = false
            }
        } else {
            in_word = true
        }
    }
    if in_word { count += 1 }
    ret count
}

@main {
    n := count_words("README.md")
    @pf("word count: {n}\n")
}
```

## Complete Example: Config Reader

```
fn read_key(path: str, key: str) -> str {
    f := @fs::FileReader::open(path) catch |_| { _ => { ret "" } }
    defer f.cl()

    while f.eof() {
        line := f.rln() catch |_| { _ => { break } }
        # lines are "key=value"
        @pf("{line}\n")
    }
    ret ""
}
```

## Directory Listing — `@fs::ls`

`@fs::ls(path)` lists the contents of a directory. Returns `?[]entry` — null if the path doesn't exist or isn't a directory.

```
files := @fs::ls("/home/user/docs")
if files != undef {
    @pf("found {} items\n", files.len)
    for e => files {
        @pf("  {} — file: {}\n", e.path(), @fs::isFile(e.path()))
    }
}
```

### Entry methods

| Method | Returns | Description |
|--------|---------|-------------|
| `e.path()` | `str` | Absolute path of the entry |
| `e.isFile()` | `bool` | True if the entry is a regular file |
| `e.isDir()` | `bool` | True if the entry is a directory |

## Simple Reader/Writer — `@fs::reader` / `@fs::writer`

A lightweight handle-based API that separates path validation from file open. Returns `?handle` — null if the path is invalid.

```
# Reader
rdr := @fs::reader::init("/etc/hostname")
if rdr != undef {
    rdr.open()
    defer rdr.cl()
    # read lines via rdr.readLine(buf)
    @pl("opened ok")
}

# Writer
wtr := @fs::writer::init("/tmp/output.txt")
if wtr != undef {
    wtr.open()
    wtr.write("hello world\n")
    wtr.cl()
}
```

### Reader methods

| Method | Description |
|--------|-------------|
| `rdr.open()` | Open the file for reading |
| `rdr.cl()` | Close the file |
| `rdr.readLine(buf)` | Read next line into `buf`; returns `?[]u8` (null at EOF) |

### Writer methods

| Method | Description |
|--------|-------------|
| `wtr.open()` | Create/truncate the file for writing |
| `wtr.cl()` | Close the file |
| `wtr.write(str)` | Write a string to the file |

> For low-level byte I/O with endian control, use the existing `@fs::FileReader` / `@fs::FileWriter` / `@fs::ByteReader` / `@fs::ByteWriter` APIs.
