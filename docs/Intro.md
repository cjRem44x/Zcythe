# Introduction

I have always wanted to make a programming language, and discovering Zig made that possible. Zig is fast, covers all possible types, and works directly with C.

The goal of Zcythe is to compile (or rather transpile) my src language into Zig. With structs, enums, static data blocks, and direct access to Zig and C libraries.

Later on I want to build in support for using with Python libs.
