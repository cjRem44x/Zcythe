Now that we have something up and running let me lay down some rules.

```
x := "hello" # this should be infered as str
@typeOf(x) # -> str , not []const u8; even thought that is what it is under the hood in Zig
```

All implicit types should be determined for there explicit fit, like auto's in C++. 
The 'any' type in Zig should be a fall back if the type is not infered.

Next thing I want is comptime generics,
```
fn foo(@comptime T t) {
    typeT = @typeOf(t)

    if typeT == i32 {do something}
}
```
So, implicit types are inferred like auto's and comptimes work like they do in Zig.
